Java 修饰符 【 アクセス修飾子 】

  • default:默认,不使用任何修饰符
  • private:同一类内可见（变量,方法）不能修饰（外部类）
  • public:对所有类可见(类,接口,变量,方法)
  • protected:对同一包内的类和所有子类可见(变量,方法)不能修饰（外部类）

自增 自减 （区别）

// 自增
public static void main(String[] args) {
        int x = 2;
        System.out.println("x:"+(x++));
    }
// x:2
public static void main(String[] args) {
        int x = 2;
        System.out.println("x:"+(++x));
    }
// x:3

// 自减
public static void main(String[] args) {
        int x = 2;
        System.out.println("x:"+(x--));
    }
// x:2
public static void main(String[] args) {
        int x = 2;
        System.out.println("x:"+(--x));
    }
// x:1

关系运算符 【 比較演算子 】

运算符 機能 例
== 判断两个数值是否相等,相等 返回True） A == B (False)
!= 判断两个数是否相等,不想等 返回True） A != B (True)
> 判断两个数值大小,左大于右 返回True） A > B (False)
< 判断两个数值大小,左小于右 返回True） A < B (True)
<= 判断两个数值大小,左小于右或者等于左 返回True） A <= B (True)
>= 判断两个数值大小,左大于右或者等于左 返回True） A >= B (False) 
// 例
 public static void main(String[] args) {
        int a = 10;
        int b = 5;
        System.out.println("A > B:"+(a>b)); // True 10>5
        System.out.println("A < B:"+(a<b)); // False 10<5
        System.out.println("A <= B:"+(a<=b)); // False 10<=5
        System.out.println("A >= B:"+(a>=b)); // True 10>=5
    }

  • extends (单一继承):一个子类只能拥有一个父类

  • implements(多继承):可以同时继承多个接口

接口 interface

interface 接口名称 extends [其他接口]{
    // 声明变量
    // 抽象方法
}

// 例子
interface Cat{
    public void eat(); //  接口方法
    public void sleep(); // 接口方法
}

// 实现
public class Mikan implements Cat{
    public void eat(){
        System.out.println("Mikan can eat!");
        // 实现
    }
    public void sleep(){ // 接口的实现
        System.out.println("Mikan can sleep!");
        // 实现
    }

    public static void main(String[] args){
        Mikan mik = new Mikan(); // 创建对象 
        mik.eat(); // 调用方法
        mik.sleep(); 
    }
}

封装 Encapsulation

メリット

  • 良好的封装可以减少耦合
  • 类内部的结构可以自由修改
  • 成员变量可以精确控制
  • 隐藏信息 实现细节
// 封装一个类
public class Cat{
    private String name;
    private Integer age; // 属性
    private float weight; // 
    // 提供对外访问的公共方法
    public String getName(){
        return name; // get 方法
    }
    public void setName(){
        this.age = age; // set 方法
    }
    public Integer getAge(){
        return age;
    }
    public void setAge(){
        this.age = age;
    }
    // ....
}
