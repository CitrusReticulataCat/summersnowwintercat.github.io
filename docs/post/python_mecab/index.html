Mecab について

モジュール方法についてのメモ、テキストから単語へ

MeCabとは：MeCabは 京都大学情報学研究科−日本電信電話株式会社コミュニケーション科学基礎研究所 共同研究ユニットプロジェクトを通じて開発されたオープンソース 形態素解析エンジンです。 言語, 辞書,コーパスに依存しない汎用的な設計を 基本方針としています。 パラメータの推定に Conditional Random Fields (CRF) を用 いており, ChaSenが採用している 隠れマルコフモデルに比べ性能が向上しています。また、平均的に ChaSen, Juman, KAKASIより高速に動作します。 ちなみに和布蕪(めかぶ)は, 作者の好物です。

pip install mecab-python3

# ipadic は単語辞書
tagger = MeCab.Tagger(ipadic.MECAB_ARGS)
print(tagger.parse("人を信じよ、しかし、その百倍も自らを信じよ。"))

名詞を引き出す

word_box =[]
for t in tagger.parse(data).split('\n'):
    if t.split('\t')[0]!='EOS' and t.split('\t')[0]!='':
        if t.split('\t')[1].split(',')[0]=='名詞':
            # 数字抜き、形容動詞語幹、副詞可能
            if t.split('\t')[1].split(',')[1]!='数' and t.split('\t')[1].split(',')[1]!='形容動詞語幹' and t.split('\t')[1].split(',')[1]!='副詞可能':
                word_box.append(t.split('\t')[0])
len(word_box)

計算結果：65

# TF 計算 （TF = count / all count）
tf_box = {}
for k,v in Counter(word_box).items():
    tf_box[k]=v/len(word_box)
list(tf_box)[0:10]
