Spring Boot

Spring Boot makes it easy to create stand-alone, production-grade Spring based Applications that you can “just run”.

We take an opinionated view of the Spring platform and third-party libraries so you can get started with minimum fuss. Most Spring Boot applications need minimal Spring configuration.

Features

  • Create stand-alone Spring applications

  • Embed Tomcat, Jetty or Undertow directly (no need to deploy WAR files)

  • Provide opinionated ‘starter’ dependencies to simplify your build configuration

  • Automatically configure Spring and 3rd party libraries whenever possible

  • Provide production-ready features such as metrics, health checks, and externalized configuration

  • Absolutely no code generation and no requirement for XML configuration

注释 作用 注释 作用
@Entity 实体类的声明（数据库实体) @Repository 数据库操作的持久化Dao层
@Table 设置表名 @Query SQL操作
@GenerateValue 主键生成策略 @RestController *
@Column 声明字段 @RequestMapping *
@Id 主键 @Data 为类提供读写属性
@Autowired 自动导入对象到类中,声明依赖 @SpringBootApplication 主类(SpringBoot的基石)
@Service 服务层（逻辑处理层) @Controller MVC控制层中的C,调用Service
@Configuration 注册额外的bean或导入其他的配置 @PostMapping POST请求
@JPAGenerate JPA 主键生成策略 @GetMapping GET请求
@Modifying 注解JPA修改操作 @PutMapping PUT请求
@Transactional 事务注解 @DeleteMapping DELETE请求
@Scope BEAN作用域 @PatchMapping PATCH请求
@Getter 提供Getter方法 @Setter 提供Setter
@ToString 提供ToString方法 @PersistenceContext 执行持久化操作
@Bean Bean组件,一般用于配置 @Component *

@GenerateValue

// 主键策略 默认为IDENTITY
public enum GenerationType{
	TABLE, // 特定表格来生成主键
	SEQUENCE, // 某些不支持主键自增的数据库 用序列作为主键
	IDENTITY, // 主键自增
	AUTO // 根据数据库自动选择其中一种策略
}

@Table,Entity,Id

@Entity
@Table(name = "tableName")
public class TableName {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    private String name;
    private String description;
    // 省略getter/setter......
}
